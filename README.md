# dev-work-wall-vuejs

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles
```
yarn run build #(& minifies for production)
yarn build --mode development
```

### Lints and fixes files
```
yarn run lint
npm run-script lint
yarn lint
```

### Run your tests (@todo)
```
yarn run test
```
